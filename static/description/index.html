Integrate Google Maps into Odoo

Show all your partners' locations on Google Maps

This module brings four new features:

  • New view map allows users to view all partners' addresses on Google Maps.
  • New widget gplaces_address_autocomplete enables Google Places Autocomplete Address Form in the partner form view,
    providing autocomplete functionality when typing an address for a partner (or any field using this widget).
  • New widget gplaces_autocomplete enables Google Places Autocomplete in the partner form view,
    providing autocomplete functionality when typing a partner's name (or any field using this widget).
  • Map Localization

New Widget Google Places Autocomplete Address Form

Provides autocomplete functionality when typing an address.

New Widget Google Places Autocomplete

Provides autocomplete functionality when typing a partner's name.

List of Google APIs & Services Required

  1. Geocoding API
  2. Maps JavaScript API
  3. Places API
